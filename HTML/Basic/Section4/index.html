The HTML is the root of the document

Inside the HTML element there are two main elements Head, Body

Body

The body element contains the content of the document, such as text, images, and other media.

Head

The head element contains metadata about the document, such as the title, character set, and links to stylesheets or scripts.

There is one more element that is outside the root html element

The doctype declaration is not an HTML element, but it is used to specify the version of HTML being used in the document.

Example of a doctype declaration

The doctype declaration for HTML5 is <!DOCTYPE html>.
